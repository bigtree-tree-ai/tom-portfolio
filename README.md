# 大树路跑跑 - 个人网站

一个创意个性化的个人介绍网站。

## 设计理念

**视觉风格**：有机流动、粒子系统、呼吸感
**核心交互**：跟随鼠标的粒子流 + 自定义光标
**色彩方案**：温暖的陶土色系 + 深邃的墨蓝背景

## 功能特性

- ✨ 粒子系统跟随鼠标流动
- 🎯 自定义光标增强交互
- 💫 页面加载动画序列
- 🌊 呼吸动画的姓名展示
- 📱 完全响应式设计

## 技术栈

- 纯 HTML/CSS/JavaScript
- 无外部依赖
- 使用 Canvas API 实现粒子效果

## 本地预览

直接在浏览器中打开 `index.html` 即可预览。

## 部署

网站部署在阿里云服务器上，通过 GitHub Webhook 实现自动更新。
